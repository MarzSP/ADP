# AlgoData (Maven)

Minimale implementatie van Maven project met Java 24 (set in `pom.xml`).

## Opdracht Implementatie van algoritmes en datastructuren
## Inleiding

Deze opdracht toetst of je de kennis die je hebt opgedaan in het vak Algoritmes, datastructuren en programmeertalen (ADP) kunt toepassen. Ben je in staat om een gegeven selectie van algoritmen te implementeren en kun je uitleggen waarom jouw implementatie snel werkt en efficient met resources omgaat en waar verbeteringen mogelijk of noodzakelijk zijn?
## Doel

Je implementeert zelf een voorgegeven verzameling algoritmes (in het nakijkmodel is na te lezen welke). Dit doe je in een programmeertaal naar keuze. Het resultaat van je werk presenteer je aan je docent. Tijdens deze presentatie leg je uit hoe je code werkt, welke keuzes je hebt gemaakt en waar je code mogelijk verbeterd kan worden. Gebruik het nakijkmodel als richtlijn voor je presentatie.
## Beoordeling en assessments

Beoordeling geschiedt aan de hand van de rubric "beoordelingsformulier algoritmen adp". Deze rubric wordt ingevuld door de docent na afloop van één of meerdere assessments waarin je laat zien hoe je code werkt en wat de kwaliteit ervan is. Omdat je meerdere algoritmes moet implementeren, is er ruimte in de lesplanning om je werk in delen te laten zien, bijvoorbeeld elke twee weken drie algoritmen.


## Generieke implementaties

Je implementaties van de algoritmes dienen zo generiek mogelijk zijn. Zo dient een lijst overweg te kunnen met allerlei verschillende typen objecten (wel moet elke lijst alleen objecten van hetzelfde typen bevatten).
#Efficiëntie

Je implementaties dienen efficiënt te zijn. "Efficiënt" slaat zowel op de snelheid (runtime) als op het gebruik van resources, zoals geheugen en schijfruimte.

## Testen

Je dient zelf zorg te dragen voor testdata waarmee je aantoont dat je implementaties correct zijn. De omvang van je dataset dient voldoende te zijn om de efficiëntie van je code aannemelijk te maken.

Uiteraard is je code voorzien van automatisch uit te voeren tests. Gebruik van een unit testing framework is toegestaan maar niet verplicht.

## Te implementeren algoritmes
- Lijsten (ArrayList, LinkedList)
- Priority Queue
- Sorteeralgoritmen: Een merge sort en een naar keuze
- Hash Table
- Binary Search
- Dijkstra's algoritme
